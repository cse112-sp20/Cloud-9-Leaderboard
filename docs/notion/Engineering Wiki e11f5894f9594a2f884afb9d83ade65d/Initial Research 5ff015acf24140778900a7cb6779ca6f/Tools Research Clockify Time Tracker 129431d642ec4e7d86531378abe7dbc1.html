Tool Name: Clockify Time Tracker

Tool Type:

Tool Overview

This Chrome Extension will count the time of the user's activity and calculate it to see the user's productivity.

Good Features

Give 3 good feature/functionality of this tool

  1. It provides many kinds of user's activity. For example like is this activity billable, is this activity a project, add tags, etc. It also provides the user with daily, weekly, and detailed data and graphs.
  1. The user can create an account for it, so, it can be use in different computers.
  1. It can calculate the time of the user's productivity outside Chrome.

Bad/Needs Improvement

Give 3 features/functionality that needs to be improved with this tool

  1. To access more features, the user's need to pay for at least $10 for regular membership and $30 for premium membership.
  1. The user needs to put the titles and description for every activities, it can be a waste of time for the user.
  1. This extension only provides a stopwatch and processes the data, so, the user needs to start the stopwatch and provides more information for it to be effective.
