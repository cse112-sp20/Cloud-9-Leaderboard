Tools Research - CodeTime

Tool Name: CodeTime

Tool Type:

VSCode Extension

Tool Overview

Code Time is an open source plugin for automatic programming metrics and time-tracking. The following metrics are tracked:

  • Active Code time
  • Lines of code added
  • Lines of code deleted
  • Characters added
  • Characters deleted
  • Total keystrokes
  • KPM

There is a Global average displayed so the user can compare himself/herself against the rest of the world.
